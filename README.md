## üöÄ Quick Start (15 Minutes)

### Step 1: Backend Setup (5 min)

```bash
cd ai-marketing-system/backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt

# Edit .env with your API keys
python main.py
```

**Backend running:** http://localhost:8000

### Step 2: Frontend Setup (5 min)

```bash
cd ai-marketing-system/frontend
npm install
npm run dev
```

**Frontend running:** http://localhost:3000

### Step 3: Start Using (5 min)

1. Open http://localhost:3000
2. Click "Content Generator"
3. Generate your first AI content
4. Click "Leads" and import customers
5. Create and send your first campaign!

**Full instructions:** [QUICKSTART.md](./ai-marketing-system/QUICKSTART.md)
